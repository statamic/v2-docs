Ensures that the string ends with a specified string. If it doesn't, it will now.

```.language-yaml
links:
  - statamic
  - wilderborn.com
```

```
{{ links }}
  <li>{{ value | ensure_right=".com" }}</li>
{{ /links }}
```

```.language-output
<li>statamic.com</li>
<li>wilderborn.com</li>
```
