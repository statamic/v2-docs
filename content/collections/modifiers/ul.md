Turn an array into an HTML unordered list element.

```.language-yaml
food:
  - sushi
  - broccoli
  - kale
```

```
{{ food | ul }}
```

```.language-output
<ul>
  <li>sushi</li>
  <li>broccoli</li>
  <li>kale</li>
</ul>
```
