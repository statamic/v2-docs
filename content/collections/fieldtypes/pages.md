## Data Structure {#data-structure}

The Pages fieldtype is a [Relate fieldtype](/reference/fieldtypes/relate), which means the Pages will be saved asÂ IDs.

``` .language-yaml
pages:
  - 892jfsd9a90as
  - 134jk1h78dfas
```

## Templating

Use the [Relate tag](/reference/tags/relate) to loop through the IDs and fetch the content data.

```
<ul>
  {{ relate:pages }}
    <li>{{ title }}</li>
  {{ /relate:pages }}
</ul>
```

``` .language-output
<ul>
  <li>How to Basic</li>
  <li>How to Extreme</li>
</ul>
```
