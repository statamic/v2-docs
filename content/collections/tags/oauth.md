## Usage {#usage}

Regular/verbose syntax. This can be useful if the provider needs to be a variable.

```
{{ oauth provider="github" }}
```

``` .language-output
/oauth/github
```

Shorthand:

```
{{ oauth:github }}
```

``` .language-output
/oauth/github
```

With a redirect parameter:

```
{{ oauth:github redirect="/account" }}
```

``` .language-output
/oauth/github?redirect=/account
```
